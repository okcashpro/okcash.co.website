Okcash para Desarrolladores

  • OK
  • para Desarrolladores

El mas sociable y sencillo sistema de pagos

A menos que los pagos deban estar asociados directamente a facturas automáticas, aceptar dinero es tan sencillo como enviar una dirección de Okcash escrita o en link, o mostrando el código bidimensional QR de nuestra dirección. Este sencillo proceso esta al alcance de cualquier usuario y llena las necesidades de un amplio rango de clientes. Cuando se realiza de manera pública, es especialmente útil para donaciones transparentes y propinas. Incluso puedes compartir Okcash a traves de IRC y diferentes Redes Sociales como Reddit y Twitter sin siquiera tener una cartera instalada.

APIs de terceros

Hay varios servicios de procesamiento de pagos de terceros que proveen de APIs; no necesitas guardar Okcash en tu servidor y manejar la seguridad que esto implica. Adicionalmente, la mayoría de estas APIs te permiten procesar facturas y cambiar tu Okcash por tu moneda local a costos competitivos. Siendo Código Libre tu también puedes utilizarlos para crear tus propios proyectos y nuevos servicios.

Puedes crear tu propio sistema financiero

Si no deseas utilizar APIs de terceros, puedes integrar un nodo de Okcash directamente a tus aplicaciones, dandote la habilidad de poder convertirte en tu propio banco y procesador de pagos. Con todas las responsabilidades que esto implica, puedes construir asombrosos sistemas con Okcash que procesen las transacciones a tu propia manera.

Direcciones Okcash para verificar transacciones

OK crea direcciones unicas para cada transacción. Así que si fueras a crear un sistema de pagos asociado a facturas, simplemente necesitas generar y monitorear una dirección de OK para cada pago. Nunca deberías de utilizar la misma dirección para realizar mas de una transacción.

La mayor seguridad se encuentra del lado del cliente

La mayor parte de la seguridad es manejada por el protocolo, eliminando la necesidad de cumplir con PCI. La prevención de fraudes puede ser simplificada al monitoreo de una simple variable: el número de confirmaciones. Más allá de eso, mantener nuestro OK seguro es principalmente cosa de asegurar nuestra cartera y utilizar HTTPS o algún otro protocolo de seguridad para enviar el requerimiento de pago a los clientes.

El Futuro de los Pagos Sociales

Okcash te permite diseñar nuevos y creativos servicios en línea que no podían existir antes por limitaciones financieras. Esto incluye los sistemas de propinas, soluciones de pago automático, servicios distribuidos de crowd-funding, manejo de pagos con seguros de tiempo, monitoreo de activos públicos, servicios de depósitos de bajo riesgo, canales de micro pagos, albercas de staking y mucho más.
