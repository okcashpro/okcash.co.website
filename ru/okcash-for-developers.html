Okcash для разработчиков

  • Okcash
  • Для разработчиков

Самая простая и самая социальная платежная система

Для принятия платежей в системе Okcash достаточно просто дать свой адрес или показать QR-код. Такая простота делает систему доступной для всех пользователей и удовлетворяет потребности большинства клиентов. Прозрачность системы и легкость использования способствуют пожертвованиям, чаевым и поощрениям. Вы даже можете поделиться монетами Okcash через IRC или социальные сети, такие как Reddit или Twitter при этом не устанавливая кошелек.

API от третьих сторон

Существует много платежных сервисов от третьих сторон, предоставляющих API; вам не нужно самому хранить Okcash на сервере и заботиться о безопасности. Кроме этого, большинство этих API позволяют обрабатывать заказы и конвертировать Okcash в вашу местную валюту по выгодному курсу. Благодаря открытому исходному коду их можно также использовать для создания новых сервисов.

Вы можете стать финансовой системой для самого себя

Если вы не хотите использовать API от третьих сторон, вы можете интегрировать узел Okcash в свои приложения; это позволит вам стать самому себе и банком, и платежным оператором. Неся всю полноту ответственности, вы сможете создавать поразительные системы, которые будут обрабатывать транзакции Okcash как вы пожелаете.

Использование адресов Okcash для отслеживания заказов

Okcash создает уникальный адрес для каждой транзакции, поэтому, если вы создаете платежную систему для обработки заказов, вам просто нужно генерировать и мониторить адреса OK для каждого платежа. Адреса Okcash следует использовать лишь один раз для одной транзакции.

Большинство решений безопасности находится на стороне клиента

Большинство решений безопасности находится в протоколе, что позволяет избавиться от стандарта PCI. Защита от мошенничества сводится к мониторингу количества подтверждений. Для безопасного хранения валюты OK нужно только обезопасить свой кошелек и использовать HTTPS или другие безопасные протоколы для отправки запросов платежей своим покупателям.

Будущее социальных платежей

Okcash позволяет создавать новые и креативные онлайн-сервисы, которые не могли существовать раньше из-за технологических ограничений. Сюда входят: система чаевых, автоматические платежи, распределенные сервисы краудфангдинга, управление платежами во времени, публичное отслеживание средств, низкорисковые услуги гарантов, каналы микроплатежей, пулы для стейкинга и многое другое.
